# Bird Blaster 🐦

Аркадная мобильная игра на Phaser 3: стреляй по птицам, собирай монеты, открывай улучшения и соревнуйся за рекорд.

## Запуск

### Python
```bash
cd bird-blaster
python3 -m http.server 8080
```
Открой: http://localhost:8080

### Node.js
```bash
cd bird-blaster
npx serve .
```
Открой: http://localhost:3000

## Управление
- Тап / клик и удержание = заряд выстрела
- Отпусти = выстрел
- Чем дольше удержание, тем сильнее и быстрее пуля

## Структура проекта
- `index.html` — точка входа
- `game.js` — вся игровая логика
- `assets/` — ассеты, птицы, фон, эффекты
- `firebase-config.js` — конфиг глобального рейтинга

## Глобальный рейтинг (Firebase)
1. Открой `firebase-config.js`
2. Вставь свои значения Firebase
3. Включи `window.BIRD_BLASTER_ENABLE_GLOBAL_LEADERBOARD = true;`
4. В Firestore создай коллекцию `leaderboard`

Если Firebase не настроен, игра использует локальный рейтинг.
